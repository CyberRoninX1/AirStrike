## ✨ Features

| Category | Capabilities |
|----------|--------------|
| **🔍 Reconnaissance** | Network scanning, client discovery, hidden SSID detection |
| **🤝 Handshake Attacks** | 4-Way Handshake capture, PMKID capture, automated deauthentication |
| **🎭 Evil Twin** | Rogue AP creation, captive portal phishing, credential harvesting |
| **🔓 WPS Attacks** | Pixie Dust brute force, WPS PIN attacks |
| **💀 Deauthentication** | Targeted deauth, broadcast deauth, custom packet rates |
| **⚡ Cracking** | Hashcat/aircrack-ng integration, distributed cracking support |
| **📊 Reporting** | HTML/JSON/PDF reports, attack logging |
| **🎨 UI** | Interactive TUI with real-time updates, colored output |
